SVG Path Meter

This page contains a giant line-lane in SVG. Length matters.

Rule: keep paths where data-ink=yes, data-band=east, and measured path length is prime.

Then sort kept paths by data-rank, decode each char via char = k XOR ((length*5 + rank*7 + 23) & 127).

It is readable and measurable in-browser. Doing hundreds manually is the trap.
