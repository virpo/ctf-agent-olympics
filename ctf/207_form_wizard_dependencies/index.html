  • Base required: team_name, contact_email, uses_api, mode, alerts
  • If uses_api=yes -> api_key is required and must start with ak_
  • If mode=advanced -> advanced_code is required (6 digits)
  • If alerts=yes -> phone is required
  • Goal: find the earliest valid submission and read its payload
